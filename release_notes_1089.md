## v1.0.89 — 원가비교 기준판매가 제거 · 관리자 일일현황 개별삭제 · 입고단가 진단

- **원가비교 탭에서 '기준판매가' 컬럼을 제거**했습니다. (중분류는 유지)
- **관리자 일일현황 개별 삭제**: 관리자 모드에서 일일현황 표의 행을 선택하고 **우클릭 → "🗑 선택 행 삭제(관리자)"** 로 개별(여러 행 가능) 삭제할 수 있습니다. 삭제 시 모든 사용자의 공유 현황에서 제거됩니다. (일반 사용자에겐 메뉴 미표시)
- **입고단가 매칭 실패 원인 안내**: 매칭이 0건이면 실제 EcountERP 응답으로 원인(권한 없음 EXP00001 / HTTP 412 등)을 팝업으로 안내합니다.

설치 후 자동 업데이트로도 적용됩니다.
