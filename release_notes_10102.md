## v1.0.102 — 업데이트 요청 / 관리자 승인 후 시스템 조회

- **일반 사용자**: 재고현황 탭에 **"🔔 업데이트 요청"** 버튼 추가. 누르면 관리자에게 갱신 요청이 접수되고 "승인 대기" 안내가 표시됩니다.
- **관리자**: 대기 중인 요청을 자동 감지(약 1분 간격)해 **"○○님이 재고현황 업데이트를 요청했습니다 — 지금 1회 조회해 전체 공유할까요?"** 팝업을 띄웁니다.
  - 승인(예) 시 그 시각 기준으로 **시스템이 재고현황을 1회 조회해 전체에 공유**하고 요청을 해제합니다.
  - 같은 요청은 한 번만 안내합니다.

설치 후 자동 업데이트로도 적용됩니다.
