## v1.0.70 — 관리자 비밀번호 변경 / 30일 주기 / 드롭다운 디자인

- **관리자도 비밀번호 변경 가능**: 관리자(THEFEELKOREA)도 우측 상단 '비밀번호 변경'으로 비밀번호를 바꿀 수 있습니다. (서버에 안전하게 저장됩니다.)
- **30일 주기 변경 안내**: 비밀번호를 변경한 지 30일이 지나면, 로그인 시 비밀번호 변경 안내가 표시되고 변경 창이 열립니다.
- **년도·월 드롭다운 디자인 개선**: 클릭은 잘 되지만 투박했던 드롭다운을, 앱과 어울리는 **둥근(라운드) 드롭다운**으로 다시 다듬었습니다. (열고 항목 클릭 시 즉시 선택)

설치 후 자동 업데이트로도 적용됩니다.
