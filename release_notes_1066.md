## v1.0.66 — 일일현황 기간 선택 / 그래프 '건' / 조회자 / 로그인 기억

### 일일현황 탭
- **년도·월 선택**: 년도(2026~2028년)와 월(전체·1~12월)을 선택해 해당 기간만 볼 수 있습니다.
- **그래프 '건' 표기**: 원가차이 추이 막대에 `42건 (32%)`처럼 건수에 '건'을 표시합니다.
- **조회자 표시**: 표 마지막 칸에 해당 비교를 실행한 **조회자 ID**를 표시합니다. (관리자 THEFEELKOREA는 '임정수'로 표기)

### 로그인
- **아이디 기억 / 비밀번호 기억** 체크박스를 추가했습니다(각각 분리). 체크하면 다음 실행 시 자동으로 채워집니다.

설치 후 자동 업데이트로도 적용됩니다.
