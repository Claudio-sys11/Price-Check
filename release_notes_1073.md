## v1.0.73 — 일일현황 23:00 자동 확정 + 사용자 관리 행 선택

### 일일현황
- **당일(00:00~22:59)에는 모든 조회 기록**이 표시됩니다(같은 날 여러 번 비교해도 전부 누적).
- **매일 23:00에 자동으로 그날 '마지막 조회 결과'를 최종으로 확정**하고, 그날의 중간 기록은 정리되어 **1건만** 남습니다. (앱이 23:00에 켜져 있을 때 동작하며, 닫혀 있었다면 다음 실행 시 과거 날짜를 자동 정리)
- 과거 날짜는 각각 최종 1건으로 유지됩니다.

### 사용자 관리
- 목록에서 **행을 클릭해 선택**하면 강조 표시되고, 선택된 사용자가 안내줄에 나타납니다.
- 선택 후 **승인 / ID 수정 / 거절 / 삭제** 버튼이 해당 사용자에게 동작합니다. (단일 선택, 줄무늬 표시로 가독성 향상)

설치 후 자동 업데이트로도 적용됩니다.
