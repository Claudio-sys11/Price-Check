## v1.0.92 — 입고단가 파일 불러오기(접속 없이 매칭)

- 재고현황 탭에 **"📄 입고단가 파일 불러오기"** 버튼을 추가했습니다.
- EcountERP에서 내보낸 **품목 파일(CSV/Excel, 1만건 이상)**을 선택하면 **품목코드·입고단가 컬럼을 자동 인식**해 입고단가를 불러옵니다.
  - 입고단가 컬럼만 선택(판매단가/출고 제외), CSV 인코딩(UTF-8/CP949) 자동 처리, 대용량 지원.
- 불러온 뒤에는 **재고현황 조회 시 접속 없이(재고 1회 접속만으로) 입고단가가 매칭**됩니다. 이미 조회해 둔 상태면 즉시 재매칭됩니다.

사용 흐름: EcountERP에서 품목 목록(품목코드+입고단가) CSV/Excel 내보내기 → 앱에서 불러오기 → 재고현황 조회.

설치 후 자동 업데이트로도 적용됩니다.
