## v1.0.75 — 계정 보안/관리 강화

### 관리자 — 사용자 관리
- **비밀번호 초기화**: 선택한 사용자의 비밀번호를 **000000**으로 초기화합니다. 사용자는 다음 로그인 시 새 비밀번호로 변경하도록 안내됩니다.
- **잠금 해제**: 잠긴 계정을 해제합니다(상태에 🔒잠김/해제요청 표시, 잠김 인원수 표시).
- **마지막 접속**: 각 사용자의 마지막 접속 일시와 경과 시간(예: 2026-06-18 12:10 (3시간 전))을 표시합니다.

### 계정 잠금
- **비밀번호를 5회 틀리면 계정이 잠깁니다.** 잠긴 사용자는 로그인 시 안내가 표시되고, **관리자에게 잠금 해제를 요청**할 수 있습니다. 로그인 성공 시 실패 횟수는 초기화됩니다.

### 내 정보 변경
- 우측 상단 **비밀번호 변경 왼쪽에 '정보 변경'** 버튼을 추가했습니다. 이름·휴대폰을 변경할 수 있으며, 관리자도 변경할 수 있습니다.

설치 후 자동 업데이트로도 적용됩니다.
