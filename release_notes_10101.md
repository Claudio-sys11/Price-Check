## v1.0.101 — 재고현황 '시스템 1회 조회 → 전체 공유' (EcountERP 접속수 절감)

기존에는 사용자마다 각자 EcountERP에 접속·조회해 접속수가 사용자 수만큼 늘었습니다. 이제:

- **관리자(=시스템)**: 재고현황을 조회하면 결과가 **자동으로 전체 공유**되고, 앱 실행 중 **매시 자동 조회·공유**합니다. (EcountERP 접속은 관리자만)
- **일반 사용자**: 직접 조회 버튼 없이 **"🔄 공유 재고현황 불러오기"**만 제공되며, 로그인 시 **공유본을 자동으로 불러와** 표시합니다(EcountERP 접속 0회). 필터·엑셀 내보내기·총수량/총금액 모두 동작합니다.
- 공유 데이터는 일일현황과 동일한 비공개 저장소(inventory_shared.json)에 저장됩니다.

→ EcountERP 접속이 (사용자 수 × 조회) → 시스템 1회로 감소합니다. (자동 공유 주기: 매시, 조정 가능)

설치 후 자동 업데이트로도 적용됩니다.
